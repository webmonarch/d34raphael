d34raphael API Documentation Added

We've added API documentation for the d34raphael specific extensions to d3.

Highlights include:

  • d3.raphael(paper)

    Initializes a d3 Raphael selector for the specified paper.

  • D3RaphaelSelection

    d34raphael's Raphael-aware selection class.

  • d3.raphael.axis()

    Raphael versions of d3's SVG axis renderers

For more information about using the raphael extensions to d3, see the examples or the detailed usage documentation.
