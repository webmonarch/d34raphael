Class D3RaphaelAxis

A Raphael axis renderer function, intended to be used like d3.svg.axis. caveates and differences documented below.

If you don't see a method documented, it probably doesn't exist yet.

NOTE: The object is actually a function with signature function(selection), to be called with a selection to which it renders it's axis,

{D3RaphaelAxis} classPrefix(val)
Get or set the class name prefix appended to the class names used to differentiate parts of the rendered axis.

Class name suffixes used internally are:
path
axis ticks
pathdomain
axis domain line (and end-ticks)
So, for example, if you specified a class name prefix xaxis_, you would want to specify CSS selectors, .xaxis_path and .xaxis_pathdomain
