(2) Write Raphael Compatible d3 Code

When using d34raphael, you must remember you aren't writing SVG...You are writing Raphael vector code.

The main differences between an SVG and Raphael version of a d3 page is:

  1. Include d34raphael supported scripts
  2. Setup the Raphael paper (instead of SVG)
  3. Call Raphael version of SVG functions (like: d3.raphael.axis() instead of d3.svg.axis())
  4. Replace SVG g references (Raphael doesn't currently have group elements)
  5. Tweak your CSS selectors (since raphael is a flat, unlike SVG)
